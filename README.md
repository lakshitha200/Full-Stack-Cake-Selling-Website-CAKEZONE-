# Full Stack Cake Selling Website (CAKEZONE)

- Backend = Spring Boot
- Frontend = React JS
- Data Base = MySQL

Designed and developed a dynamic, responsive full stack cake selling website (CAKEZONE)  using Spring Boot, React Js and MySQL.
Enhanced user experience with intuitive, user-friendly features. Used Spring leated concepts.
